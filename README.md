# LRS SaaS API Market Service

A comprehensive SaaS platform for Learning Record Store (LRS) API marketplace.

## Features

### User Features
- User Authentication (Sign up, Login, SSO)
- My Page (Dashboard, Profile Management, Payment Management)
- Subscription Plans (Basic, Standard, Professional, On-Premise)
- API Marketplace (Learning Analytics, Standard Integration, Learning Tools)
- Notice Board
- 1:1 Support
- FAQ

### Admin Features
- User Management
- Order Management
- Payment Management
- Subscription Management
- API Management
- Dashboard Management
- Board Management
- Popup Management
- Notification Management
- Statistics
- System Management

## Tech Stack
- React 18
- TypeScript
- Material-UI
- Redux Toolkit
- React Router
- Chart.js
- Axios

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm start
```

3. Build for production:
```bash
npm run build
```
